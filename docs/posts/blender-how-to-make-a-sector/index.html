[Blender] How to Make a Sector

记录一下如何快速生成扇形

  1. add a circle

  2. add a plane

  3. enter the edit mode, subdivide the plane for 10~20 times

  4. back to object mode, add a curve modifier to the plane, and use the circle as curve object

  5. change the sector like the following gif, then apply to keep the shape
