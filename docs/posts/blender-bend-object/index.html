[Blender] How to Bend Object

记录一下Blender中弯曲物体的简单方法

  1. select the bend-needed object, then enter the edit mode

  2. use shortcut “ctrl+r” to make some loop cut

  3. back to object mode, then add an arrows as the origin of bending. add->empty->arrows

  4. select the bend-needed object, add a modifier-simple Deform

  5. settings are as follows

    • select bend type
    • angle: control the bend angle, modify the number to change object’s bending angle
    • origin: use the arrows created in step.3
    • axis: choose which axis you want to bend along

  6. apply the modifier, now you get a bended object
